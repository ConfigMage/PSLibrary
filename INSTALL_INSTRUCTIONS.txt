================================
POWERSHELL SCRIPT LIBRARY
Installation Instructions
================================

Thank you for downloading PowerShell Script Library!
This is a script management tool with a VS Code-inspired interface.

QUICK START - WINDOWS
=====================
1. Extract this folder to your desired location (e.g., C:\Programs\PSLibrary)
2. Navigate to the 'deploy' folder
3. Double-click 'install_windows.bat'
4. Follow the prompts
5. Once complete, use the desktop shortcut or run 'run.bat'

QUICK START - MAC/LINUX
=======================
1. Extract this folder to your desired location
2. Open Terminal and navigate to the PSLibrary folder
3. Run: cd deploy && chmod +x install_unix.sh && ./install_unix.sh
4. Once complete, run: ./run.sh

REQUIREMENTS
============
- Windows 10/11, macOS 11+, or Linux (Ubuntu 20.04+)
- Python 3.10 or higher
- 200MB free disk space

IF INSTALLATION FAILS
=====================
1. Make sure Python 3.10+ is installed:
   - Windows: Download from https://www.python.org/downloads/
   - Mac: brew install python@3.12
   - Linux: sudo apt install python3 python3-pip

2. Try manual installation:
   - Open Command Prompt/Terminal in the PSLibrary folder
   - Run: pip install -r requirements.txt
   - Then run: python main.py

WHAT THIS APPLICATION DOES
==========================
- Organize PowerShell and Batch scripts in folders
- Syntax highlighting for easy reading
- Search across all your scripts
- VS Code-inspired dark theme
- No cloud storage - all data stays local

FIRST TIME USE
==============
1. Click "New Folder" to create categories
2. Right-click folders to add scripts
3. Double-click scripts to edit
4. Ctrl+S to save your changes

SUPPORT
=======
If you encounter issues:
1. Run 'check_installation.py' to diagnose problems
2. Check the deploy/README.md for troubleshooting
3. Ensure antivirus isn't blocking the application

Enjoy organizing your scripts!
